Privacy Policy for KLU Attendance Extension

This extension is designed to help KLU students calculate and track their attendance using data from the KLU ERP system.

Data Collection:
The extension does not collect, store, or share any personal or sensitive user data.

Data Usage:
The extension temporarily reads attendance-related information (such as course names, conducted classes, and attended classes) from the KLU ERP attendance page in order to perform calculations and display insights to the user.

Permissions:
- activeTab: Used to access the current ERP page only when the user interacts with the extension.
- scripting: Used to extract attendance data from the page.
- host permissions:to https://newerp.kluniversity.in/ for accessing attendance information.

Data Storage:
No user data is stored locally or remotely.

Third-Party Sharing:
The extension does not share any data with third parties.

User Control:
All functionality is triggered only when the user interacts with the extension.
